# Starter line templates per style

Empty patterns to fill in. Each preserves the style's prosody contract — register, line length, syllable density.

---

## پاپ (Pop)

**Verse template (6–10 syllables, colloquial)**

```
[noun phrase] + [verb] + [object]
example: «تو هنوز توی فکرمی»

[time phrase] + [emotional reaction]
example: «دیشب ازت دلم گرفت»
```

**Chorus template (8 syllables × 4 lines, hook word at start)**

```
[hook] + [3 syllables] + [object]
[hook] + [3 syllables] + [object]
[hook] + [variant 3 syllables] + [emotion]
[hook] + [variant 3 syllables] + [resolution]

example with hook = هنوز:
هنوز این شب صدای توئه
هنوز این در نگاهِ توئه
هنوز اینجام ولی نیستی
هنوز قلبم با ردِّ توئه
```

---

## رپ (Rap)

**Verse template (12–16 syllables, broken into 2–3 internal phrases)**

```
[short clause], [short clause], [longer clause with internal rhyme]
example: «شب رفت، صبح اومد، ولی این دل هنوز بیداره»
```

**Hook template (4 short lines, 4–6 syllables each)**

```
[2 words]
[2 words]
[2 words echoing line 1]
[2 words punch]

example:
رفتی و رفتم
موندم و سوختم
شب موند و من موند
دل هم نموندم
```

**Internal rhyme pattern**

In each line, place 2–3 rhyming syllables internally. Example: «شب-ـه شهر-ـه دل / هـر شب می‌میره».

## راک (Rock)

**Verse template (8–12 syllables, attack syllable on beat 1)**

```
[strong-onset word] + [continuation]
example: «شب میاد و شهرِ ما تنهاست»

[declarative statement]
example: «من نخواستم این همه دور بشم»
```

**Chorus template (4 lines, declarative, monosyllabic hook)**

```
[hook] + [continuation]
[hook] + [variant]
[hook] + [intensification]
[final declaration]

example with hook = نه:
نه دیگه برنمی‌گردم
نه دیگه نمی‌سوزم
نه این صدا، نه این دل
دیگه چیزی نمیخوام
```

## سنتی / آوازی (Traditional)

**Bayt template (formal, 7–11 syllables)**

```
[formal subject] + [formal verb] + [classical imagery]
example: «دلم در پای تو افتاده ای جان~~»
```

**Refrain template (single hemistich repeated)**

```
[hook phrase ending on open vowel]~~
example: «بیا ای آرامِ جان~~»
```

---

## تصنیف (Tasnif)

**Stanza template (4 lines, formal, ending with refrain)**

```
[story beat 1]
[story beat 2 ending with rhyme A]
[story beat 3]
[story beat 4 ending with rhyme A]

[refrain — verbatim every time]
```

---

## بالاد سینمایی (Cinematic ballad)

**Verse template (8–14 syllables, semi-formal, breath room)**

```
[image] + [pause indicated by line break] + [reaction]
example:
«چراغِ این اتاق روشن
ولی صدا هیچی نمیگه»
```

**Chorus template (4–6 lines, slow build)**

```
[hook] + [expansive phrase 1]
[hook] + [expansive phrase 2]
[bridge phrase]
[resolution]
[optional 5th-6th line — expand or repeat]
```

---

## مینیمال شاعرانه (Minimal poetic)

**Stanza template (3–4 short lines, lots of space)**

```
[image — 4 syllables max]
[reaction — 5 syllables max]
[pause — empty line or just «...»]
[recurring phrase]

example:
چراغ خاموش
صدا تموم
...
هنوز اینجام
```

---

## الکترونیک (Electronic)

**Verse template (4–8 syllables, very low density)**

```
[short phrase]
[shorter phrase]
[repeat of phrase 1]
[variation]

example:
یه شب دیگه
بی تو دوباره
یه شب دیگه
دیگه نمیشه
```

**Chorus template (1–2 lines repeated)**

```
[hook line]
[hook line]
[hook line — slight variation]
[hook line]
```

---

## نسل جدید (New generation)

**Verse template (5–9 syllables, conversational)**

```
[everyday observation]
[emotional pivot]
example:
«یه روز یه جا یه چیزی گفت
از اون به بعد دلم گرفت»
```

**Chorus template (2 lines repeated, very direct)**

```
[hook line, 6–8 syllables]
[answer line, 6–8 syllables, end-rhyme]

example:
هنوز اینجام، هنوز هستم
ولی بی تو شکستم
```

---

## How to use these templates

1. Pick the style.
2. Copy the template for that style.
3. Fill the slots with words from the song's image bank.
4. Run the AI-singer cleanup checklist.
5. Read aloud. Rewrite anything that doesn't sing.

Templates are scaffolding, not constraints. Once the lyric works, it doesn't need to look like the template anymore.
