# Style prompt — fill-in skeleton

Copy this, fill the bracketed slots, paste into Suno's **Style** field. Keep it to ~25–40 descriptors. Comma-separated. No periods.

```
[Language/genre — e.g. Persian alternative folk],
[overall atmosphere — e.g. cinematic emotional],
[texture — e.g. warm analog tape texture],
[vocal character — e.g. intimate male vocal, fragile whisper],
[mix style — e.g. soft tape saturation, vinyl noise],
[primary instrument 1 — e.g. minimal acoustic guitar],
[primary instrument 2 — e.g. kamancheh textures],
[primary instrument 3 — e.g. distant ney flute],
[dynamics — e.g. emotional slow build],
[mood — e.g. nostalgic, melancholic, hopeful],
[production realism — e.g. realistic human imperfections, no over-polish],
[reference vibe — e.g. indie soundtrack feeling, non-commercial production],
[chorus character — e.g. deep emotional chorus, repeatable hook],
[vocal phrasing — e.g. natural Persian vocal phrasing]
```
