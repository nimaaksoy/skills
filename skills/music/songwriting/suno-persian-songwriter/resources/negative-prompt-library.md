# Negative prompt library

Use these in Suno's **More Options → Negative Prompts** field. Pick the ones that match the song's risk profile — don't ship the entire list. 6–10 negatives is usually enough.

## Anti-commercial pop

```
generic pop,
happy commercial vibe,
radio jingle energy,
corporate background music,
upbeat advertising sound
```

## Anti-EDM / festival

```
EDM festival drop,
big-room synth lead,
sidechain pumping,
trance arpeggios,
DJ build-up
```

## Anti-trap / overprocessed

```
cheap trap beat,
808 sub bass overload,
aggressive autotune,
robotic vocal pitch,
trap hi-hat triplets
```

## Anti-cheap mix

```
muddy mix,
overly bright mastering,
harsh treble,
clipping distortion,
washed-out reverb
```

## Anti-genre-drift

```
random genre switching,
inconsistent vocal style,
abrupt mood change,
unrelated instrument swap
```

## Anti-generic vocal

```
overprocessed vocals,
pop diva vocal,
breathy whisper turned shouty,
auto-tuned chorus,
karaoke phrasing
```

## Anti-cliché Persian-pop

```
late-2000s Persian pop synth,
4-on-the-floor pop drums,
saccharine string section,
overly sweet melody,
LA studio polish
```

## Mixing

Combine ~2 lists for a balanced ban. Example for a Persian indie folk piece:

```
generic pop,
happy commercial vibe,
EDM festival drop,
overprocessed vocals,
aggressive autotune,
muddy mix,
random genre switching
```
